The Game: One Piece Monopoly
My game is a brilliant, real-time multiplayer web application built as a midterm project that merges classic Monopoly mechanics with the One Piece universe.

Core Features & Mechanics:

The Board: A custom-designed 40-space grid. Instead of standard Monopoly streets and railroads, players navigate through iconic One Piece islands, purchase pirate ships, and draw themed action cards.

The Economy: All currency transactions, property purchases, and rent calculations are handled in Berries.

Real-Time Multiplayer: Players can join and interact seamlessly. This is powered by real-time database management, allowing the game to instantly sync turn management, dice rolls, movement calculations, and property ownership across all players' screens.

Tech Stack: The user interface and game logic are driven by React, while Firebase works behind the scenes as your database to handle the real-time syncing and state management.
